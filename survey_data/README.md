# Survey Dataset: 28 Multi-view Images for 3DGS Preprocessing

## 📌 Dataset Overview
This dataset is specifically curated for benchmarking preprocessing strategies in the RGB-based 3D Gaussian Splatting (3DGS) pipeline. It consists of 28 high-quality multi-view images captured around a single object to evaluate SfM (Structure-from-Motion) stability and reconstruction fidelity.

## 📱 Data Acquisition Information
- **Capture Device**: Samsung Galaxy Z Flip 4 (Main Camera)
- **Image Format**: PNG
- **Total Images**: 28 frames
- **Scene Type**: 360-degree object-centric capture
- **Resolution**: 1600 x 900
